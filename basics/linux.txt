к любой команде можно дописать --help и выйдет описание как использовать эту команду

cd test    - переходим  в папку text (если эта папка существует)


cd ..      - переход в родительскую папку
cd.        - переход в эту же папку (никуда не перейдем)
cd         - переход в домашнюю папку
cd ~       - переход в домашнюю папку (~)
pwd        - показывает абсолютный путь до текущей папки 
mkdir test      - создание новой папки test
mkdir test/new_dir      - создание новой папки new_dir в новой уже существующей папке test 
touch test.txt          - создание файл test.txt 
rm test.txt             - удаление файла test.txt
rm -rf  test              - удаление папки test со всем её содержимым
cat                     - просмотр содержимого фацла linux.txt 
ls test                     - просмотр всех файлов и папок  внутри папки test 
ls -a test                  - просмотр всех файлов и папок внутри папки test включая скрытые 
ls -l                       - просмоттр всех папок с более подробной информацией
